# FinTechPaymentSystem

Included in this repo is my implementation of a minimal FinTech-style payments system composed of two Spring Boot microservices. The domain is simple internal wallet transfers: move money between two accounts and keep an accurate ledger even under failures and heavy concurrency. Based on requirements found in the PDF.
